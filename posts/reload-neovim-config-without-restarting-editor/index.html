Reload Neovim Config Without Restarting Editor

Posted on Jan 22, 2021

Just watched neovim video on youtube at ThePrimeagen channel. And he can install plugin without close/restarting the nvim editor, then i realized what command he typed.

Ok, i got it.

:so %

That’s it! And i find explained of the command from superuser.com.

:so for short command of :source which reads the content of the specified file and treats it as nvim/vim code. And % stads for current file name.
